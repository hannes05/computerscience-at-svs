Year 7 Research Page – Operating Systems

1. What is an Operating System?

An Operating System (OS) is the main software that controls the computer. It manages the hardware and allows users to run other programs such as games, browsers, and word processors.

  • Examples: Windows, Mac OS, Linux, Android, iOS
🔗 Learn more on BBC Bitesize

2. Hardware vs Software

Term Meaning Example
Hardware Physical parts of the computer Keyboard, screen, mouse
Software Programs that tell the computer what to do Games, browsers, Word
🔗 Hardware vs Software (BBC Bitesize)

3. What Does an OS Do?

  • 🎨 Provides a user interface (GUI or CLI)
  • 🧠 Handles memory and multitasking
  • 🖱️ Controls hardware devices
  • 🗂️ Manages files and security
🔗 Interactive tutorial (GCF Global)

4. Types of User Interface

GUI (Graphical User Interface): Uses pictures, icons, and windows.
CLI (Command Line Interface): Uses typed text commands.

🔗 GUI vs CLI (BBC Bitesize) ▶️ Watch: GUI vs CLI Explained

5. Device Drivers

A device driver is a small program that helps the OS communicate with devices like printers, mice, and cameras. Without the correct driver, the device might not work properly.

🔗 Learn more: Device Drivers (HowStuffWorks)

6. Booting Up

When you turn on your computer, it performs a boot routine — checking hardware and loading the OS.

  • Cold boot: Turning the computer on
  • Warm boot: Restarting the computer
🔗 What is Booting? (Techopedia)

7. Desktop vs Mobile Operating Systems

  • Desktop: Windows, Mac OS, Linux
  • Mobile: Android, iOS
